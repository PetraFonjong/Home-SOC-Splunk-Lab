 Home SOC Lab – macOS Log Analysis with Splunk

A Security Operations Center (SOC) home lab project that demonstrates collecting, ingesting, and analyzing macOS system logs in Splunk while simulating attacker activity from a Kali Linux virtual machine. This project showcases log analysis, threat detection, dashboard creation, and basic incident investigation techniques.

---



The objective of this project was to build a small SOC environment capable of detecting suspicious activity using Splunk Enterprise.

A Kali Linux virtual machine was used to simulate attacker behavior, including network reconnaissance with Nmap. Logs generated by both macOS and Kali were collected and analyzed to identify security events.

This project demonstrates practical cybersecurity skills that are commonly used by Security Operations Center (SOC) analysts.
